This are the GITHUB tips

  1. Introducir en la terminal "git status"
  2. Introducir en la terminal "git add ." para añadir todo --- Aquí solo se está preparando el archivo, pasandolo a stage
  3. Introducir en la terminal "git commit -m "..."" --- Aquí se ha confirmado el cambio pero en el repositorio local, para verlo en GitHub hace falta un cuarto paso
  4. Introducir en la terminal "git push" --- Aquí FINALIZA el proceso

De esta manera hemos finalizado el proceso de modifing, stagind and commiting changes to our GitHub.
